# Changelog

## 1.0.0

Initial release.

### Features

- **10 conversion categories, 70 units** - Length, Weight, Temperature,
  Volume, Area, Speed, Time, Data, Energy, and Pressure.
- **Live conversion** - type in a custom numeric keypad and see every other
  unit in the category update instantly, no "convert" button needed.
- **Tap-to-swap** - tap any result to make it the new "from" unit.
- **Favorites** - star any from/to unit pair for one-tap access later from
  the Saved tab.
- **Recents** - the Home tab surfaces recently used categories (can be
  turned off in Settings).
- **Customization** - accent color (8 choices), light/dark/system theme,
  category card style, card corner roundness, decimal precision
  (significant figures), thousands-place grouping, and haptic feedback,
  all in Settings and persisted on-device.
- **Onboarding** - a short first-run walkthrough, plus a persistent "How to
  use this app" screen reachable any time from Settings.
- **No accounts, no ads, no dummy data** - a fresh install starts empty;
  everything you see is either instructional content or something you
  added yourself.
- **Privacy Policy & Terms of Use** - included in-app and as static pages
  in `docs/`, describing the app's use of Firebase Analytics for
  anonymous, aggregate usage insights (no ads, no advertising ID
  collection).
