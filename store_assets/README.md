# Store assets

Assets for the Play Console listing itself, not bundled into the app.

- `play_store_icon_512.png` - the 512x512 "app icon" image Play Console
  asks for when setting up the store listing (Grow → Store presence →
  Main store listing). This is separate from the in-app launcher icon
  (`android/app/src/main/res/mipmap-*/ic_launcher*.png`).
